Demo seed photos
================

Placeholder photos for the auto-generated demo dataset (DemoDataService),
sourced via the Openverse API (https://openverse.org), which aggregates
Creative-Commons and public-domain images. Used for educational/demo display
only; real user reports use uploaded photos. com.lfms.util.SeedImageFactory
maps each item to one of these files and falls back to a generated icon if a
file is missing. Credits per image:
